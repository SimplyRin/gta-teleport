# GtaTeleport
GTA っぽいテレポートを再現する Paper プラグイン

参考: https://twitter.com/hookuru_/status/2066780862507798978

- /gtatp x y z: 指定した座標へ移動します。
- /gtatp bed: 最後に寝た場所、またはリスポーン地点に移動します。
- /gtatp targetPlayer: 指定したプレイヤーの場所へ移動します。
